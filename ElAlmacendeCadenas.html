Código Fuente 
class Receta {
  constructor(nombre) {
    this.nombre = nombre;
    this.items = [];
    this.tiempo = 0;
    return this;
  }

  agregar(item, cant) {
    this.items.push({item, cant});
    return this;
  }

  tiempo(min) {
    this.tiempo = min;
    return this;
  }

  build() {
    return this;
  }
}

Concepto Clave

Cada método retorna this para permitir el encadenamiento. El orden: inicializar → configurar → construir.

Demo: SweetAlert2 con Method Chaining

SweetAlert2 usa el patrón de Method Chaining para configurar alertas. Observa cómo encadenamos métodos: 

const Swal = swal;

// Sin chaining (verbose)
Swal.fire("Título");
Swal.setTitle("Nuevo título");
Swal.showConfirmButton(false);

// Con Method Chaining (SweetAlert2 estilo)
Swal.fire({
  title: "¡Hola!",
  text: "Mensaje con chaining",
  icon: "success",
  confirmButtonText: "Aceptar"
})
