Seguridad de tipos (simulada)

En Swift, los enums con valores asociados garantizan que cada caso lleve los datos correctos. Aquí lo simulamos con objetos JavaScript que tienen una propiedad tipo y datos (precio, emoji). Al usar async/await, obtenemos un objeto bien definido; en el enfoque tradicional, podríamos recibir cualquier cosa o perder el contexto.

Ejemplo enum (Swift) enum Fruta {
  case manzana(precio: Double)
  case banana(oro: Bool)
  case naranja(vitaminaC: Int)
}
