SwiftyAlert

Simplifica UIAlertController con Builder Pattern, async/await y enums.

🎯 Motivación ⚙️ Method Chaining ⏳ Async/Await 🔐 Enums
🚀 Motivación

Crear alertas en UIKit tradicional implica mucho código repetitivo: inicializar UIAlertController, configurar estilos, añadir acciones con closures. SwiftyAlert envuelve todo en una interfaz fluida y expresiva.

Objetivo: Pasar de 10+ líneas a una sola llamada encadenada.

🔨 Builder Pattern (Method Chaining)

Cada método (ej. .title(), .message(), .addButton()) retorna self, permitiendo encadenar configuraciones. Internamente se va construyendo un objeto que al final presenta la alerta.

SwiftyAlert()
    .title("¡Hola!")
    .message("Mundo")
    .addButton("Aceptar")
    .show()
⏳ Async/Await en alertas

En lugar de closures anidados, SwiftyAlert usa async para esperar la respuesta del usuario de forma lineal. El código se lee como si fuera síncrono.

let opcion = await SwiftyAlert()
    .title("Confirmar")
    .addButton("Eliminar", role: .destructive)
    .addButton("Cancelar")
    .show()

if opcion == .eliminar { ... }
🔐 Enums para resultados

En lugar de índices mágicos, SwiftyAlert devuelve un enum con casos descriptivos: .ok, .cancel, .delete, etc. Esto da seguridad y claridad al manejar la respuesta.

enum AlertResult {
    case ok, cancel, delete
    case custom(String)
}

// Función que muestra un modal y retorna una promesa
function showModernAlert(title, message, buttons) {
  return new Promise((resolve) => {
    // (Implementación interna: asigna texto al modal y muestra)
    document.getElementById('modalTitle').textContent = title;
    document.getElementById('modalMessage').textContent = message;
    const btnContainer = document.getElementById('modalButtons');
    btnContainer.innerHTML = '';
    buttons.forEach(({ text, value, variant }) => {
      const btn = document.createElement('button');
      btn.className = `btn ${variant === 'dark' ? 'btn-swifty' : 'btn-outline-swifty'} mx-1`;
      btn.textContent = text;
      btn.onclick = () => {
        bootstrap.Modal.getInstance(document.getElementById('demoModal')).hide();
        resolve(value);
      };
      btnContainer.appendChild(btn);
    });
    new bootstrap.Modal(document.getElementById('demoModal')).show();
  });
}

// Uso con async/await
document.getElementById('btnAsync').addEventListener('click', async () => {
  const resultado = await showModernAlert('Confirmar', '¿Quieres continuar?', [
    { text: 'Cancelar', value: 'cancel', variant: 'outline' },
    { text: 'Aceptar', value: 'ok', variant: 'dark' }
  ]);
  document.getElementById('resultDemo').innerText = `Resultado: ${resultado}`;
});
