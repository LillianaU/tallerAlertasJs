Ejemplo 3: YAML / TOML

Definición

YAML

Formato de serialización de datos legible por humanos, usa indentación. Ideal para archivos de configuración.

TOML

Formato de configuración simple, similar a INI pero con especificación estricta. Mejor para configuraciones complejas.

1. YAML - Ejemplos

Configuración de servidor
servidor: host: localhost puerto: 8080 debug: true bd: nombre: miapp usuarios: - admin - guest
Listas y objetos
productos: - nombre: Laptop precio: 1000 stock: 5 - nombre: Mouse precio: 25 stock: 100

2. TOML - Ejemplos

Configuración de app
[server] host = "localhost" port = 8080 [database] name = "miapp" enabled = true [database.options] timeout = 30
Arrays y tablas
frutas = ["manzana", "banana"] [persona] nombre = "Juan" edad = 30 [[usuarios]] nombre = "Ana" rol = "admin"
