Ejemplo 2: Destructuring & Spread

Definición

Destructuring

Permite extraer valores de arrays u objetos en variables individuales con una sintaxis concisa.

Spread Operator (...)

Permite expandir elementos de un iterable (arrays, objetos) en lugares donde se esperan múltiples elementos.

1. Destructuring de Arrays

Básico
const colores = ['rojo', 'verde', 'azul']; const [primero, segundo, tercero] = colores;
Saltar elementos
const nums = [1, 2, 3, 4, 5]; const [a, , , d] = nums;
Valores por defecto
const [a, b, c = 10] = [1, 2];

2. Destructuring de Objetos

Básico
const persona = { nombre: 'Ana', edad: 30, ciudad: 'Madrid' }; const { nombre, edad } = persona;
Renombrar variables
const persona = { nombre: 'Ana', edad: 30 }; const { nombre: nombreCompleto, edad: años } = persona;
Valores por defecto
const { nombre, pais = 'España' } = { nombre: 'Juan' };
Objetos anidados
const data = { usuario: { perfil: { edad: 25 } } }; const { usuario: { perfil: { edad } } } = data;

3. Spread en Arrays

Combinar arrays
const arr1 = [1, 2, 3]; const arr2 = [4, 5, 6]; const combinado = [...arr1, ...arr2];
Copiar array
const original = [1, 2, 3]; const copia = [...original];
Agregar elementos
const base = [2, 3, 4]; const nuevo = [1, ...base, 5];

4. Spread en Objetos

Combinar objetos
const obj1 = { a: 1, b: 2 }; const obj2 = { b: 3, c: 4 }; const merged = { ...obj1, ...obj2 };
Copiar objeto
const original = { nombre: 'Ana', edad: 30 }; const copia = { ...original };
Actualizar propiedades
const usuario = { nombre: 'Ana', edad: 30 }; const actualizado = { ...usuario, edad: 31, ciudad: 'Madrid' };

5. Parámetros Rest (...args)

Función con argumentos variables
function suma(...numeros) { return numeros.reduce((a, b) => a + b, 0); } suma(1, 2, 3, 4, 5);
Destructuring con rest
const nums = [1, 2, 3, 4, 5]; const [primero, segundo, ...resto] = nums;

6. Ejemplos de Vida Real

Gestión de configuración
const configDefault = { theme: 'light', lang: 'es', timeout: 3000 }; const userConfig = { theme: 'dark' }; const finalConfig = { ...configDefault, ...userConfig };
Actualizar estado de React
const estado = { usuario: 'Ana', rol: 'admin' }; const nuevoEstado = { ...estado, rol: 'superadmin' };
Form con datos de usuario
const formData = { nombre: '', email: '' }; const update = (field, value) => ({ ...formData, [field]: value });
Carrito de compras
const carrito = [{ id: 1, cantidad: 2 }]; const agregar = (producto) => [...carrito, producto]; const actualizar = (id, cant) => carrito.map(p => p.id === id ? {...p, cantidad: cant} : p);
