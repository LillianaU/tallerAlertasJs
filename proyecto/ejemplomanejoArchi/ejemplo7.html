¿Qué es fs y Buffer?

fs (File System) Node.js

Módulo de Node.js para trabajar con archivos:

  • fs.readFileSync() - Leer archivos
  • fs.writeFileSync() - Escribir archivos
  • fs.appendFileSync() - Agregar contenido
  • fs.unlinkSync() - Eliminar archivos
  • fs.mkdirSync() - Crear carpetas
Buffer Node.js

Clase para manejar datos binarios:

  • Buffer.from(str) - Crear desde texto
  • Buffer.alloc(size) - Crear buffer vacío
  • buffer.write(str) - Escribir en buffer
  • buffer.toString() - Convertir a texto
  • buffer.toString('hex') - Convertir a hex

Ejercicio 1: Buffer Básico Node.js

¿Qué hace? Crea un buffer desde texto y muestra los bytes (códigos ASCII).

const buffer = Buffer.from('Hola Mundo', 'utf8'); console.log(buffer); // <Buffer 48 6f 6c 61 20 4d 75 6e 64 6f> console.log(buffer.toString()); // Hola Mundo console.log(buffer.length); // 11 const bufferVacio = Buffer.alloc(10); console.log(bufferVacio); // <Buffer 00 00 00 00 00 00 00 00 00 00>

Ejercicio 2: Escribir en Buffer Node.js

¿Qué hace? Escribe texto en posiciones específicas del buffer.

const buf = Buffer.alloc(16); buf.write('Hello', 0, 'utf8'); buf.write(' World', 5, 'utf8'); console.log(buf.toString()); // Hello World console.log(buf.slice(0, 5).toString()); // Hello

Ejercicio 3: fs - Crear Archivo Node.js

¿Qué hace? Crea un archivo en el sistema de archivos.

const fs = require('fs'); const contenido = 'Hola, esto es un archivo de prueba'; fs.writeFileSync('mi_archivo.txt', contenido); console.log('Archivo creado'); // Verificar const existe = fs.existsSync('mi_archivo.txt'); console.log('Existe:', existe); const stats = fs.statSync('mi_archivo.txt'); console.log('Tamaño:', stats.size, 'bytes');

Ejercicio 4: fs - Leer Archivo Node.js

¿Qué hace? Lee el contenido de un archivo.

const fs = require('fs'); // Síncrono const contenido = fs.readFileSync('mi_archivo.txt', 'utf8'); console.log(contenido); // Como Buffer const buffer = fs.readFileSync('mi_archivo.txt'); console.log(buffer);

Ejercicio 5: Streams con Chunks Node.js

¿Qué hace? Procesa archivos grandes en chunks (trozos) sin cargar todo en memoria.

const fs = require('fs'); const stream = fs.createReadStream('mi_archivo.txt', { encoding: 'utf8', highWaterMark: 1024 }); let chunks = 0; stream.on('data', (chunk) => { chunks++; console.log('Chunk:', chunk.length, 'bytes'); }); stream.on('end', () => { console.log('Total chunks:', chunks); });

Ejercicio 6: fs - Agregar Contenido Node.js

¿Qué hace? Agrega contenido a un archivo sin borrar lo anterior.

const fs = require('fs'); fs.writeFileSync('log.txt', 'Línea 1\n'); fs.appendFileSync('log.txt', 'Línea 2\n'); fs.appendFileSync('log.txt', 'Línea 3\n'); const contenido = fs.readFileSync('log.txt', 'utf8'); console.log(contenido);

Ejercicio 7: CRUD con Archivos Node.js

¿Qué hace? Create, Read, Update, Delete con archivos JSON.

const fs = require('fs'); // CREATE let usuarios = [{id:1, nombre:'Juan'},{id:2, nombre:'María'}]; fs.writeFileSync('usuarios.json', JSON.stringify(usuarios)); // READ let datos = JSON.parse(fs.readFileSync('usuarios.json', 'utf8')); console.log('Leer:', datos); // UPDATE datos[0].nombre = 'Juan Actualizado'; fs.writeFileSync('usuarios.json', JSON.stringify(datos)); // DELETE fs.unlinkSync('usuarios.json'); console.log('Eliminado');

Ejercicio 8: fs - Directorios Node.js

¿Qué hace? Crea, lista y gestiona directorios.

const fs = require('fs'); // Crear directorio if (!fs.existsSync('micarpeta')) { fs.mkdirSync('micarpeta'); } // Crear archivo dentro fs.writeFileSync('micarpeta/archivo.txt', 'contenido'); // Listar const archivos = fs.readdirSync('micarpeta'); console.log('Archivos:', archivos); // Info const stats = fs.statSync('micarpeta/archivo.txt'); console.log('Tamaño:', stats.size);

Ejercicio 9: Buffer - Bytes y Hex Node.js

¿Qué hace? Convierte texto a bytes y hexadecimal.

const buf = Buffer.from('Hola'); // Bytes console.log('Bytes:', buf); // [72, 111, 108, 97] // Hexadecimal console.log('Hex:', buf.toString('hex')); // 486f6c61 // Base64 console.log('Base64:', buf.toString('base64')); // SG9sYQ== // Desde hex const buf2 = Buffer.from('486f6c61', 'hex'); console.log('Desde hex:', buf2.toString()); // Hola

Ejercicio 10: Comparación String vs Buffer Node.js

¿Qué hace? Compara la longitud de strings con buffers.

const texto = 'A'.repeat(10000); const buffer = Buffer.from(texto); console.log('String length:', texto.length); // 10000 console.log('Buffer length:', buffer.length); // 10000 // En ASCII son iguales, pero en UTF-8 pueden variar
