Ejemplo 6: Protobuf / Avro

Definición

Protocol Buffers (Protobuf)

Sistema de serialización binaria de Google, requiere esquemas (.proto). Muy rápido y eficiente.

Avro

Sistema de Apache, usa esquemas JSON. Permite evolución de esquemas dinámicamente.

1. Protobuf - Esquemas

Definición de mensaje
syntax = "proto3"; message Persona { string nombre = 1; int32 edad = 2; string email = 3; repeated string telefonos = 4; }
Mensajes anidados
message Pedido { int32 id = 1; Persona cliente = 2; repeated Item items = 3; } message Item { string producto = 1; int32 cantidad = 2; double precio = 3; }

2. Avro - Esquemas

Esquema en JSON
{ "type": "record", "name": "Persona", "fields": [ {"name": "nombre", "type": "string"}, {"name": "edad", "type": "int"}, {"name": "email", "type": ["string", "null"]} ] }
Tipo array y map
{ "type": "record", "name": "Empresa", "fields": [ {"name": "nombre", "type": "string"}, {"name": "empleados", "type": {"type": "array", "items": "string"}}, {"name": "config", "type": {"type": "map", "values": "string"}} ] }
