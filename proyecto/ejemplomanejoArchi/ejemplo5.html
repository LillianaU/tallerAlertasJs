Ejemplo 5: MessagePack / BSON

Definición

MessagePack

Formato binario eficiente para serialización, más compacto que JSON. Ideal para transmisión de datos.

BSON

Binary JSON usado por MongoDB, soporta más tipos de datos incluyendo fechas y binarios.

1. MessagePack - Ejemplos

Serialización básica
const datos = { nombre: "Ana", edad: 30 }; // Representación binaria simulada const bytes = stringToBytes(JSON.stringify(datos));
Tipos soportados
// integer, float, string, array, // map, boolean, nil, binary const tipos = { entero: 42, cadena: "hola", array: [1, 2, 3] };

2. BSON - Ejemplos

Tipos especiales BSON
// ObjectId, Date, Binary, Regex, Code const doc = { _id: ObjectId('...'), fecha: ISODate('2024-01-01'), binary: Binary('...'), regex: Regex('pattern', 'i') };
Documento MongoDB
{ _id: ObjectId("..."), nombre: "Juan", fecha_registro: ISODate("..."), tags: ["admin", "user"] }
